## ADDED Requirements

### Requirement: 本地回测记录列表加载
系统必须扫描特定的本地结果保存目录（例如 `results/`），并将其中的回测结果文件（如 JSON 或 PKL 报告）解析为包含执行时间、策略名称和交易品种的列表供用户选择。

#### Scenario: 用户浏览历史回测日志
- **WHEN** 用户导航至历史记录页面
- **THEN** 系统展示一个以时间倒序排列的下拉列表或数据表，列出所有已持久化保存的过往回测运行记录

### Requirement: 历史结果读取与复现
当用户选中某条具体的历史回测记录时，系统必须读取相关的持久化数据，并在界面上复现回测完成时的净值曲线图表和绩效指标表格，而无需重新运行回测计算过程。

#### Scenario: 用户点开特定历史记录的详情
- **WHEN** 用户从列表中点击一条名为 "2026-02-15_Turtle_SPY" 的回测记录
- **THEN** 系统加载对应的报告数据，并展示其资金曲线和详细的策略绩效统计表
