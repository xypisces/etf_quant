## ADDED Requirements

### Requirement: 应用程序主框架
系统必须提供一个基于 Streamlit 的 Web 应用程序主界面，包含侧边栏导航和主内容区域，用于集成和调度所有子模块界面。

#### Scenario: 用户启动并访问主界面
- **WHEN** 用户通过命令行启动 Dashboard 应用并通过浏览器访问
- **THEN** 系统显示带有左侧导航菜单（包含各个功能模块入口）的主界面，默认展示系统概览页

### Requirement: 统一的主题与样式
界面的视觉呈现必须保持统一，支持 Streamlit 默认的亮暗色主题切换，确保图表（Plotly）的配色方案与系统主题协调。

#### Scenario: 用户切换系统主题
- **WHEN** 用户在 Streamlit 设置中切换界面的亮暗主题
- **THEN** 界面上的图表背景和线条颜色能够自动适配新主题，保证良好的可读性
